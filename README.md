# Double Win

Gestionnaire de scores pour jeux en temps réel.

## Fonctionnalités

- Gestion de scores pour 2 joueurs
- Personnalisation complète (couleurs, polices, bordures)
- Interface web accessible via navigateur
- Système de mise à jour automatique

## Installation

Téléchargez l'exécutable depuis les releases GitHub.

## Utilisation

Lancez l'application et ouvrez les liens vers les pages Joueur 1 et Joueur 2 dans votre navigateur.
